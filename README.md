# Golden Dandelion Indicator
Displays particles on mobs effected by the Golden Dandelion.
---
![Golden Dandelion Indicator on a baby Sheep.](https://cdn.modrinth.com/data/E6BdxvsQ/images/75b569f1578b24a348c69d8034626b65ea68ee7d.png)
### Compatibility
- Only required on client-side
- No server-side installation required for multiplayer
### Version Support
- Minecraft 26.1
- Fabric and NeoForge
### Installation
1. Install [Fabric Loader](https://fabricmc.net/use/) or [NeoForge](https://neoforged.net/).
2. Place the mod `.jar` in your `mods` folder.
3. Launch Minecraft with Fabric/NeoForge.
